1. В этом репозитории хранятся некоторые задачи, которые я решал в течение университетской учебы. ВСЕ эти задания были выполнены без использования ИИ.
2. Условия задач, если они были предоставлены, находятся в соответствующих папках.
3. Чтобы запустить программы, находящиеся в папках с названиями HW_n, используйте Makefile, затем $./a.out...
4. Конкретный список аргументов и правила запуска можно найти в условиях задач, если таковые были предоставлены.
5. Для проверки работы программ, работающих с деревьями, приложен архив, содержащий файл nv_1000000.txt, представляющий из себя миллион объектов класса "student", реализованного в этих задачах.
6. Если условия не было, то, скорее всего, в самой программе явно прописаны правила использования.
7. Задачи в папках HW_n представляют из себя реализацию и некоторые манипуляции со структурами данных: списками и различными деревьями.
