В этом репозитории хранятся некоторые задачи, которые я решал в течение университетской учебы. ВСЕ эти задания были выполнены без использования ИИ.
Условия задач, если они были предоставлены, находятся в соответствующих папках.
Чтобы запустить программы, находящиеся в папках с названиями HW_n, используйте Makefile, затем $./a.out...
Конкретный список аргументов и правила запуска можно найти в условиях задач, если таковые были предоставлены.
Для проверки работы программ, работающих с деревьями, приложен файл nv_1000000, содержащий миллион объектов класса "student", реализованного в этих задачах.
Если условия не было, то, скорее всего, в самой программе явно прописаны правила использования.
Задачи в папках HW_n представляют из себя реализацию и некоторые манипуляции со структурами данных: списками и различными деревьями.
